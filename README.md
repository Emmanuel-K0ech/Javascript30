## How to Use

To run any project:
1. Clone this repository:  
   ```bash
   git clone https://github.com/your-username/javascript30-challenge.git
   ```
2. Navigate to the project's directory:  
   ```bash
   cd day-X
   ```
3. Open the `index.html` file in your browser.
